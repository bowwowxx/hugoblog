(開篇)Akitio Node之老舊MacbookPro外接顯卡大作戰(EGPU)
Jan 28, 2017
1 minute read

安迪兒手上的MacBook Pro是13吋2012年版本的老舊筆電了
趁過新年整理一下，再次強化它，提升他的效能。

先上結論

安迪兒手上這台2012年版13吋Macbook Pro升級後配備如下:
1. 雙SSD做raid 0(拆掉光碟機) 2. 16G RAM 3. 外接nvida顯卡 4. 新的筆電電池

早期Macbook Pro還不錯，耐用且可以自己隨意的更換零件
第1、2、4項安迪兒之前就換了
但顯卡和cpu這塊一直沒辦法有很好的方式做升級

mac的痛一直是顯卡不夠力

謎之音： 我也想用mac打電動呀~ＸＤ

關於顯示卡，國外的網友有一些修改方式
例如:
* 犧牲無線網卡，拆機拉線出來接轉接卡再接顯卡
* 找個pcie外接盒，剪線改電源，再接顯卡

直到最近，EGPU、電競的流行，專門的顯卡外接盒出現在市場上
安迪兒發現最近有一款ATIKIO Node剛推出上市
蠻符合安迪兒的需求，Power 400W、Thunderbolt 3
差不多市面上的高級顯卡都能跑
重點是不用改硬體線路!!!
於是…
一個箱子就出現在辦公室，不小心踢到
首先來開箱一下

安迪兒分析後
優點:
1. 有送一條1千多的Thunderbolt 3 的線 2. 有風扇散熱 3. 有把手方便整盒提起來

缺點:
1. 比想像中大 2. 有點重 3. 有風扇免不了有聲音 4. Thunderbolt3的線…只有2016後新的mac才能用
(加買一條3轉2的線要2千元…太坑了)

以上開箱完畢!
再來正式說明一下

如何修改MAC外接顯卡

開始前…很重要!!!!

  • 顯卡的跑分資料:
    http://www.videocardbenchmark.net/gpu_list.php

  • mac上ATIKIO Node盒子的顯卡支援程度:
    https://en.wikipedia.org/wiki/CUDA

  • mac升級至最新的Sierra 10.12.3 (相容較多的顯示卡)

安迪兒在顯卡這邊踩到雷，用了GTX1060
但在mac上做了很多的修改和測試
頂多只能捉到卡，但沒辦法驅動
後來查了很多資料看了人家的討論才知道
在mac上外接顯卡NVIDA只支援到Maxwell和之前的架構
新的Pascal系列，目前不支援(可以參考上面的連結列表)

以上在windows10有支援不用擔心
mac可用bootcamp方式安裝win10

1. 關掉MAC System Integrity Protection

MAC有SIP的機制，防止我們去對系統做深層的修改。
所以要先把 SIP 關掉。
步驟如下
重開機 按住command+R 進回復模式 打開終端機
輸入

csrutil disable


如果指令不能用的話
可能和安迪兒一樣磁區有點不同
這時可以做一隻mac的安裝碟
用安裝碟開機進終端機執行指令就可以了

安裝碟的製做方式
參考安迪兒以前寫的文章
自製Mac OS X El Capitan 10.11 USB安裝碟&修復RAID功能

2. 解鎖Thunderbolt 3外接顯卡

mac上其實有限制非特定的Thunderbolt硬體不給用
先接上盒子接上轉接線，看一下硬體明細 Thunderbolt那邊外接盒如果有Unsupported字眼的
可以參考 > https://github.com/KhaosT/tb3-enabler

捉下tb3-enabler，然後執行

chmod +x tb3-enabler.py
sudo tb3-enabler.py apply

看一下狀態
tb3-enabler.py status

最後restart筆電

3. 寫入mac相關硬體資訊&安裝相關的driver

可以參考
> https://github.com/goalque/automate-eGPU

捉下來後

chmod +x automate-eGPU.sh
sudo ./automate-eGPU.sh

像安迪兒是nvida
其實也可以自己去安裝相關的mac driver
或是automate-eGPU安裝過程問詢問你
(都選y)
裝完後，再按更新就可以了

  • nvidia在mac上的driver是web driver
    QUADRO & GEFORCE MACOS DRIVER RELEASE 367.15.10.35

  • 順便裝一下cuda
    CUDA

這張是mac有捉到顯卡但沒driver的

有支援的話會像這樣子

4. 以上都完成的話就可以關機，接好外接螢幕，重開機嘍

順利有支援的話，畫面就會出現了

5. 蓋上筆電單獨用外接顯卡的螢幕也ok

6. 看一下相關的資訊

7. 跑一下分吧

真的有差~XD

8. 玩一下cs，有點順XD

老舊mbp再戰2年吧~xd

以上是安迪兒macbook pro外接顯卡的記錄，收工嘍!
