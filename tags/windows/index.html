微軟windows10更新擁抱linux系統-win10 ubuntu shell

微軟在今年3月Build 2016開發者大會，拋出了一個震撼彈 就是windows10竟然可以使用ubuntu的bash shell 而且不是虛擬機器，是原生的! (微軟馬上洗白變好人了~XD) 這實在太有話題了，微軟當時有說今年夏季會提供相關更新 直到最近這陣子，安迪兒發現有新的win10 Preview版本 Link: Windows 10 Insider Preview Build 14316 Download Windows 10 build 14316 ISO images 這太有趣了，安迪兒立馬試玩一下~XD 記錄一下win10開啟ubuntu shell的方式 1. 開啟開發人員功能模式 2. 控制台->程式和功能->程式集->windows功能 勾選[windows subsystem for linux] 3. 執行bash 直接用尋找bash並執行它 (download&安裝 Bash on ubuntu on windows) 4. 搜尋&執行 裝好的Bash on Ubuntu on Windows 噢！！！！！ 太開心了吶~XD 原來在shell裡面windows的東西在這，很特別的感覺~XD win10 build 14316真是太帥氣了吶~~
